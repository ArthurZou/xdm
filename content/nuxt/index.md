# Nuxt 框架介绍

Nuxt 是一个基于 Vue.js 的全栈框架，让开发者能够构建生产级的 Web 应用。它将 Vue.js 的灵活性与服务端渲染、文件系统路由、自动导入等强大功能结合在一起。

## 核心特性

- **文件系统路由** — `pages/` 目录下的 `.vue` 文件自动生成对应路由，无需手动配置
- **多种渲染模式** — 支持服务端渲染（SSR）、静态站点生成（SSG）、客户端渲染（CSR）及混合渲染
- **自动导入** — 组件、组合式函数（composables）、工具函数全部自动导入，无需手写 `import` 语句
- **模块生态** — 丰富的官方及社区模块（Auth、Content、Image、i18n 等）开箱即用
- **TypeScript 原生支持** — 全面的类型安全，无需额外配置
- **Nitro 服务端引擎** — 高性能的服务端引擎，支持部署到任意平台（Vercel、Cloudflare、Node.js 等）

## 为什么选择 Nuxt

相比纯 Vue.js 开发，Nuxt 解决了以下痛点：

1. **SEO 友好** — 通过 SSR/SSG 让搜索引擎能够正确抓取页面内容
2. **性能优化** — 自动代码分割、懒加载、图片优化等
3. **约定大于配置** — 遵循约定即可快速上手，减少样板代码
4. **全栈能力** — `server/` 目录可直接编写 API 路由，无需额外后端
